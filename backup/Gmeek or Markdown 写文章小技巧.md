# 1. 文章内标题跳转

跳转链接：
`[跳转链接文本](#标题名称)`

跳转到的位置（可以是 # ## ### 等）：
`# 标题名称`

其中 标题名称 需满足以下规则：
- 标题文本中的空格替换为连字符 -（例如 #section 1 → #section-1）。
- 自动忽略标点符号（如 !、? 等）。
- 所有字母转换为小写（例如 #Section Two → #section-two）47。

如
`[跳转到标题3](#3.-未想好写什么abc)`
[跳转到标题3](#3.-未想好写什么)
